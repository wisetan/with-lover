MVP原则: 最小可行产品，聚焦核心陪诊场景

4. 关键业务流程
4.1 主预约流程
text
用户选择时间 → 发起咨询 → 沟通细节 → 创建订单 → 
支付保证金 → 服务执行 → 状态跟踪 → 完成评价
4.2 费用处理规则
yaml
费用规则:
  保证金: 30元
    目的: 防止爽约
    退还条件: 服务正常完成
    扣除条件: 用户无理由取消
    
  医疗费: 用户自行支付
    推荐方式: 家属远程支付
    备用方案: 陪诊师垫付+现场报销
    
  服务费: 线下协商转账
    协商时机: 咨询阶段
    支付时机: 服务完成后
